# example network

An igraph example network

## Usage

``` r
example_network
```

## Format

An igraph object

- genetic:

  Genetic sex for each sample with levels F or M

- site:

  Site for each sample with levels A, B, or C

- name:

  Name for each node

- degree:

  Degree for each node

- lat:

  Latitude for each node

- long:

  Longitude for each node

- wij:

  Weight for each edge

## Source

Simulation via ergm package
