# plot_ergm_bic

For each ergm model plots the BIC

## Usage

``` r
plot_ergm_bic(ergms, type = 1)

plot_ergm_bic(ergms, type = 1)
```

## Arguments

- ergms:

  list of ergms fitted to IBD net

- type:

  which plot to show

## Value

ggplot of BIC for each ergm

plot of fits
