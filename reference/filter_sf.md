# Filter SF object

Filter SF object

## Usage

``` r
filter_sf(sf_obj, xmin, xmax, ymin, ymax)
```

## Arguments

- sf_obj:

  object with node and edge df

- xmin:

  left long border

- xmax:

  right long border

- ymin:

  bottom lat border

- ymax:

  top lat border

## Value

filter SF object
