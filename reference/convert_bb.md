# Given a bounding box, increase the lat and long by given amounts

Given a bounding box, increase the lat and long by given amounts

## Usage

``` r
convert_bb(bb, x_adj = 0.1, y_adj = 0.1)
```

## Arguments

- bb:

  bounding box (left, bottom, right, top)

- x_adj:

  lon adjust

- y_adj:

  lat adjust

## Value

new bounding box
